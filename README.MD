## 🖥️ Projeto

Esse é um projeto web responsivo de uma cápsula do tempo para exibir minhas memórias em linha do tempo.

## 🚀 Tecnologias

Esse projeto foi desenvolvido durante o NLW da Rockeatseat com as seguintes tecnologias:

- HTML 🦴
- CSS  🥰
- Git e Github 🎮
